How to Serialize-(marshalling) Object in Ruby

October 26th, 2010

It happens many times that we want to store data on the persistance storage for using it some time later

Marshalling

Marshalling is the process by which we can serialize  and unserialize ruby object in our ruby programs. The data that is stored can be read and the original object can be reconstituted. So, In marshalling we are actually recording the data(state + code) of the object in such a way that when it is unmarshalled it can be retained into original object with its data.A more general explanation given here

class UserData

 def []=(kind, value)
   user_data = self.find_by_kind(kind.to_s) || self.new(:kind => kind.to_s)
   user_data.value = Base64.encode64(Marshal.dump(value))
   user_data.save 
 end

 def self.[](kind)
  user_data = UserData.find_by_kind(kind.to_s)
  user_data ? Marshal.load(Base64.decode64(user_data.value)) : nil
 end
end
